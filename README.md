# Todo-Graphql

## Usage
>- npm install

>- npm start

>- open <a href="http://localhost:3000/">localhost:3000 </a> in browser

#### This uses <a href="https://github.com/graphql/graphiql">Graphiql IDE</a> to run queries , mutations and communicating with the graphql server
